Motivation

When writing a new blog post on the assessment of quality of fitted Self-Organizing Map (SOM) to data I stumbled upon a problem on how to find all R packages that are related to a particular subject on CRAN. In particular, my problem was to find all packages that might contain any items related to the assessment of SOM.

Librarian Package

After goggling for a while I found the library(librarian) package which contains a function browse_cran() to search package names and descriptions within CRAN. For example, if you can’t remember what the colorbrewer package is actually called, you just type browse_cran("colorbrewer").

I decided to test it and to find any packages that are related to the Self Organizing Maps topic. I found out that I cannot use the argument som to find any items as all words containing some are also returned. A quote organizing map should work.

Actually, there is a function CRAN_package_db() from the library(tools) which obtains information about current packages in the CRAN package repository, and their check status.

There are some outliers but as the latest sensible dates are from the same day when this analysis is created then it is safe to assume that packages.rds file is up-to-date. Hence, this approach can be used to find out more about R Packages in CRAN directly from RStudio!
